I frequently find myself ranting at people that the god of Python is TypeType, and what an erratically vicious god he is.

This rarely seems to be coherent to anyone.

New strategy, inspired by finding another oddity during today’s hacking: I’m going to catalogue, as I remember or find them, all of the oh-my-god-why-would-a-language-allow-you-to-do-that things in Python.

#!/usr/bin/python2.7
#-*- coding: utf-8 -*-
#
# wtf001.py
# ---------
# Call a bound method of a class without initialising the class.
#

import types

class C:
    """This only works on old-style classes.

    Classic classes or "old-style" classes should have never existed 
    to begin with.
    """
    def __init__(s):
        print "C.__init__() called."
    def p(s, t):
        print "C.p() called."
        print t

if __name__ == "__main__":
    wtf = types.InstanceType(C)
    msg = "I am a bound method, called without calling my initialisation"
    msg += " function C.__init__() first."
    wtf.p(msg)

which results in:
